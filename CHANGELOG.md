## [v0.2.0] - 2026-03-14

### Highlights

- CLI tool for headless browser operations with `gorod fetch` command
- Cookie extraction with conversion to standard `http.Cookie` format
- Wait helper functions for reliable element interaction

### Added

- `cmd/gorod` CLI tool with `fetch` command for headless page rendering ([`e5685d9`](https://github.com/grokify/gorod/commit/e5685d9))
- `Cookies` type wrapping Rod cookies with `HTTPCookies()` and `String()` methods ([`8bef8fd`](https://github.com/grokify/gorod/commit/8bef8fd))
- `ForegroundBrowser.Cookies()` method to extract browser cookies ([`5cb78a2`](https://github.com/grokify/gorod/commit/5cb78a2))
- `WaitVisible()` function to wait for element visibility with timeout ([`c0bb044`](https://github.com/grokify/gorod/commit/c0bb044))
- `WaitClickable()` function to wait for element to be visible and enabled ([`c0bb044`](https://github.com/grokify/gorod/commit/c0bb044))
- `ErrBrowserNotInitialized` error for nil browser handling ([`5cb78a2`](https://github.com/grokify/gorod/commit/5cb78a2))

### Changed

- Use `bufio.Reader` for input handling in `ForegroundBrowser` (fixes empty line issues) ([`5cb78a2`](https://github.com/grokify/gorod/commit/5cb78a2))
- Add `MustWaitStable()` after page navigation for better reliability ([`5cb78a2`](https://github.com/grokify/gorod/commit/5cb78a2))

### Dependencies

- Update `github.com/grokify/mogo` to v0.73.5 ([`176e381`](https://github.com/grokify/gorod/commit/176e381))
- Add `github.com/spf13/cobra` for CLI framework ([`176e381`](https://github.com/grokify/gorod/commit/176e381))
- Add `github.com/jessevdk/go-flags` for example CLI parsing ([`176e381`](https://github.com/grokify/gorod/commit/176e381))
- Add `fetchup` replace directive for rod v0.116.2 compatibility ([`176e381`](https://github.com/grokify/gorod/commit/176e381))

### Documentation

- Expanded README with quick start guide, API reference, and usage examples ([`176e381`](https://github.com/grokify/gorod/commit/176e381))
- Add `examples/cookies/` demonstrating cookie extraction ([`f52c544`](https://github.com/grokify/gorod/commit/f52c544))
